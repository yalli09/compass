How to Use Compass Map

This page explains every feature of the Compass Map application.

Adding & Editing Points

  • Click on the "Add Point" section in the sidebar to create a new location.
  • You can search for an address (e.g. "980 Broadway, NYC"). The app will geocode it using OpenStreetMap's Nominatim service.
  • Optionally specify a day number; leave blank to mark the point as unscheduled.
  • You may also add a description or a photo URL to make notes about the location.
  • Existing points can be edited by clicking on their entry in the list or by right‑clicking the map marker.
  • Latitude/longitude fields are hidden by default but are preserved when editing. If you change the address the coordinates are re‑geocoded.

Filtering & Calendar

The calendar along the top of the sidebar shows the days up to the "Max Days" setting. Clicking a day filters the map to only show points assigned to that day. Click the same day again to clear the filter.

Auto‑Scheduling / Organizer

Use the Organize > Auto‑Schedule Unscheduled menu item to automatically assign unscheduled points to days. The algorithm groups points by geographic proximity, so nearby locations are clustered together rather than spread out randomly. You can configure the maximum number of days via Organize > Settings.

Note: the "Max Days" setting is shared across all users and devices. When someone updates it, everyone else will see the calendar adjust in real time.

Import & Export

Use the File menu to import or export your point list as JSON. Imported data should follow the same structure as the exported file.

Mobile & Layout

The sidebar can be toggled using the ☰ button on small screens. The layout is responsive and should work on phones and tablets.

Real‑time Sync

Multiple browser windows/tabs and users will see updates instantly thanks to the built‑in Socket.IO synchronization.

Privacy Policy - on this app

The Compass Map application stores all data locally on the server in a points.json file. There is no authentication, so anyone with access to the server can view and modify points.

Address searches and reverse lookups are performed through the OpenStreetMap Nominatim service. No additional information is sent or stored by Compass beyond the query you enter and the resulting coordinates.

When you use the import/export features, the JSON is handled entirely within your browser and the server; it is not forwarded to any third parties.
