How to Use Compass Map

This page explains every feature of the Compass Map application.

Adding & Editing Points

  • Click on the Add Point section in the Map Points tab to create a new location.
  • Enter a point name and search for an address (e.g. "980 Broadway, NYC"). The app will geocode it using OpenStreetMap's Nominatim service.
  • Optionally specify a day number; leave blank to mark the point as unscheduled.
  • Add a description to make notes about the location and a photo URL to attach an image.
  • Existing points can be edited by clicking on their entry in the list or by right‑clicking the map marker.
  • Latitude/longitude fields are hidden by default but are preserved when editing. If you change the address the coordinates are re‑geocoded.

Search & Filter Points

  • Use the search bar above the points list to find locations by name, description, address, or ID.
  • Left-click any point in the list to center the map on it and view its details.
  • Right-click any point in the list to open the edit modal.
  • The × button removes a point from the list.

Calendar & Day Filtering

The calendar along the top of the sidebar shows the days up to the Max Days setting. Clicking a day filters the map to only show points assigned to that day. Click the same day again to clear the filter.

Smart Map Display

  • When you first open Compass, the map automatically zooms to display all your points in a perfect fit.
  • When filtering by day or searching, the map updates to show only matching points.
  • In collaborative sessions, adding new points doesn't reset your current map view—work undisturbed!

Task List

Switch to the Task List tab to manage your to-do items and deadlines. Features include:

  • Create a new task with a title and optional due date.
  • Check off tasks as you complete them (they'll appear with strikethrough text).
  • View all your tasks organized with their due dates.
  • Delete tasks you no longer need.
  • All tasks are synced in real-time across all users and devices using Socket.IO.

Auto‑Scheduling / Organizer

Use the Organize > Auto‑Schedule menu item to automatically assign unscheduled points to days. The algorithm groups points by geographic proximity, so nearby locations are clustered together rather than spread out randomly. You can configure the maximum number of days via Organize > Settings.

Note: The Max Days setting is shared across all users and devices. When someone updates it, everyone sees the calendar adjust in real time.

Import & Export

Use the File menu to import or export your point list as JSON, or clear all points. Imported data should follow the same structure as the exported file. This makes it easy to backup, share, or migrate your location data.

Mobile & Layout

The sidebar can be toggled using the ☰ button on small screens. The layout is fully responsive and optimized for phones, tablets, and desktops. Use the Tools menu to switch between Map Points and Task List tabs.

Real‑time Sync

Multiple browser windows, tabs, and users will see updates instantly thanks to the built‑in Socket.IO synchronization. Both map points and tasks are synced in real-time, so changes made by any user appear immediately for everyone.

Privacy Policy

The Compass Map application stores all data locally on the server in a points.json file. There is no authentication, so anyone with access to the server can view and modify points.

Address searches and reverse lookups are performed through the OpenStreetMap Nominatim service. No additional information is sent or stored by Compass beyond the query you enter and the resulting coordinates.

When you use the import/export features, the JSON is handled entirely within your browser and the server; it is not forwarded to any third parties.
